You are given an array `points` representing integer coordinates of some points on a 2D-plane, where `points[i] = [xi, yi]`.

The cost of connecting two points `[xi, yi]` and `[xj, yj]` is the **manhattan distance** between them: `|xi - xj| + |yi - yj|`, where `|val|` denotes the absolute value of `val`.

Return *the minimum cost to make all points connected.* All points are connected if there is **exactly one** simple path between any two points.

**Example 1:**
```
Input: points = [[0,0],[2,2],[3,10],[5,2],[7,0]]
Output: 20
Explanation:

We can connect the points as shown above to get the minimum cost of 20.
Notice that there is a unique path between every pair of points.
```
**Example 2:**
```
Input: points = [[3,12],[-2,5],[-4,1]]
Output: 18
```
**Constraints:**
- `1 <= points.length <= 1000`
- `-106 <= xi, yi <= 106`
- All pairs `(xi, yi)` are distinct.

## Detailed Intuition

- Model the problem as nodes and edges, then choose BFS/DFS/Topological traversal based on dependency direction.
- Track visited/state to prevent repeated processing and cycles.
- Build the answer while traversing (order, count, or connectivity result).

**Time Complexity** : O(n)<br>
**Space Complexity** : O(n)

```java
class Solution {
    public int minCostConnectPoints(int[][] dots) {

        int size = dots.length;
        List<Point> points = new ArrayList<>();
        for (int[] dot : dots) points.add(new Point(dot[0], dot[1]));

        PriorityQueue<Distance> minHeap = new PriorityQueue<>();
        for (int i = 0; i < size; i++)
            for (int j = 0; j < points.size(); j++)
                if (i != j)
                    minHeap.add(new Distance(points.get(i), points.get(j)));

        DisjointSet set = new DisjointSet();
        int cost = 0, mergeCount = 0;
        while (!minHeap.isEmpty() && mergeCount + 1 < size) {

            Distance remove = minHeap.remove();
            if (set.union(remove.start, remove.end)) {
                mergeCount++;
                cost = cost + remove.distance;
            }
        }
        return cost;
    }
    class DisjointSet {
        Map<Point, Node> nodes = new HashMap<>();
        private void makeSet(Point val) {
            nodes.putIfAbsent(val, new Node(val));
        }
        private Node findSet(Node node) {
            if (node.parent == node) return node.parent;
            node.parent = findSet(node.parent);
            return node.parent;
        }
        public Point find(Point val) {
            makeSet(val);
            return findSet(nodes.get(val)).val;
        }
        public boolean union(Point one, Point two) {
            makeSet(one);
            makeSet(two);
            Node left = findSet(nodes.get(one)),
                 right = findSet(nodes.get(two));
            if (left == right) return false;

            if (left.rank < right.rank) {
                left.parent = right;
            } else if (left.rank > right.rank) {
                right.parent = left;
            } else {
                left.parent = right;
                right.rank++;
            }
            return true;
        }
    }
    class Distance implements Comparable<Distance> {
        Point start, end;
        int distance;
        public Distance(Point start, Point end) {
            this.start = start;
            this.end = end;
            this.distance = Math.abs(this.start.x - this.end.x) + Math.abs(this.start.y - this.end.y);
        }
        @Override
        public int compareTo(Distance that) {
            return Integer.compare(this.distance, that.distance);
        }
    }
    class Point {
        int x, y;
        public Point(int x, int y) {
            this.x = x;
            this.y = y;
        }
        @Override
        public int hashCode() {
            return Objects.hash(this.x, this.y);
        }
        @Override
        public boolean equals(Object obj) {
            Point that = (Point) obj;
            return this.x == that.x && this.y == that.y;
        }
    }
    class Node {
        Point val;
        int rank = 1;
        Node parent;
        public Node(Point val) {
            this.val = val;
            this.parent = this;
        }
    }
}
```
