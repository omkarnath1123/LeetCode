Given strings `s1`, `s2`, and `s3`, find whether `s3` is formed by an **interleaving** of `s1` and `s2`.

An **interleaving** of two strings `s` and `t` is a configuration where `s` and `t` are divided into `n` and `m` substrings respectively, such that:
- `s = s1 + s2 + ... + sn`
- `t = t1 + t2 + ... + tm`
- `|n - m| <= 1`
- The **interleaving** is `s1 + t1 + s2 + t2 + s3 + t3 + ...` or `t1 + s1 + t2 + s2 + t3 + s3 + ...`

**Note:** `a + b` is the concatenation of strings `a` and `b`.

**Example 1:**
```
Input: s1 = "aabcc", s2 = "dbbca", s3 = "aadbbcbcac"
Output: true
Explanation: One way to obtain s3 is:
Split s1 into s1 = "aa" + "bc" + "c", and s2 into s2 = "dbbc" + "a".
Interleaving the two splits, we get "aa" + "dbbc" + "bc" + "a" + "c" = "aadbbcbcac".
Since s3 can be obtained by interleaving s1 and s2, we return true.
```
**Example 2:**
```
Input: s1 = "aabcc", s2 = "dbbca", s3 = "aadbbbaccc"
Output: false
Explanation: Notice how it is impossible to interleave s2 with any other string to obtain s3.
```
**Example 3:**
```
Input: s1 = "", s2 = "", s3 = ""
Output: true
```
**Constraints:**
- `0 <= s1.length, s2.length <= 100`
- `0 <= s3.length <= 200`
- `s1`, `s2`, and `s3` consist of lowercase English letters.

**Follow up:** Could you solve it using only `O(s2.length)` additional memory space?

## Detailed Intuition

- Model the problem as nodes and edges, then choose BFS/DFS/Topological traversal based on dependency direction.
- Track visited/state to prevent repeated processing and cycles.
- Build the answer while traversing (order, count, or connectivity result).

**Time Complexity** : O(n)<br>
**Space Complexity** : O(n)

```java
class Solution {
    Map<Integer, Map<Pair, Boolean>> cache;
    public boolean isInterleave(String one, String two, String target) {

        cache = new HashMap<>();
        return canCreate(one, 0, two, 0, target, 0);
    }
    private boolean canCreate(String one, int oneIndex, String two, int twoIndex, String target, int targetIndex) {
        if (target.length() == targetIndex && one.length() == oneIndex && two.length() == twoIndex) return true;
        if (target.length() == targetIndex) return false;

        Pair pair = new Pair(oneIndex, twoIndex);
        cache.putIfAbsent(targetIndex, new HashMap<>());
        if (cache.get(targetIndex).containsKey(pair))
            return cache.get(targetIndex).get(pair);

        boolean canCreate = false;
        if (oneIndex < one.length() && one.charAt(oneIndex) == target.charAt(targetIndex))
            canCreate = canCreate || canCreate(one, oneIndex + 1, two, twoIndex, target, targetIndex + 1);
        if (twoIndex < two.length() && two.charAt(twoIndex) == target.charAt(targetIndex))
            canCreate = canCreate || canCreate(one, oneIndex, two, twoIndex + 1, target, targetIndex + 1);

        cache.get(targetIndex).put(pair, canCreate);
        return canCreate;
    }
    class Pair {
        int oneIndex, twoIndex;
        public Pair (int oneIndex, int twoIndex) {
            this.oneIndex = oneIndex;
            this.twoIndex = twoIndex;
        }
        @Override
        public int hashCode() {
            return Objects.hash(oneIndex, twoIndex);
        }
        @Override
        public boolean equals(Object obj) {
            Pair that = (Pair) obj;
            return this.oneIndex == that.oneIndex && this.twoIndex == that.twoIndex;
        }
    }
}
```
