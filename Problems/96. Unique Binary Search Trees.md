Given an integer `n`, return _the number of structurally unique **BST'**s (binary search trees) which has exactly_ `n` _nodes of unique values from_ `1` _to_ `n`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2021/01/18/uniquebstn3.jpg)
```
Input: n = 3
Output: 5
```
**Example 2:**
```
Input: n = 1
Output: 1
```
**Constraints:**

-   `1 <= n <= 19`

**Solution:**

- [x] [[Utils/Dynamic Programming]]

## Detailed Intuition

- Define DP state so each state captures the minimum information required to continue transitions.
- Write transition from previously solved states and compute in valid order.
- Initialize base cases carefully since all later states depend on them.

**Time Complexity** : O(n)<br>
**Space Complexity** : O(n)

```java
    Integer[] store;
    public int numTrees(int n) {
        store = new Integer[19];
        return backTrack(n - 1);
    }
    private int backTrack(int i) {
        if (i == 0 || i == -1)// i = 0, one node tree AND i = -1, null node tree
            return 1;
        if (store[i] != null)
            return store[i];
        
        int comb = 0;
        for (int k = 0; k <= i; k++)
            comb = comb + backTrack(k - 1) * backTrack(i -  k - 1);
        
        store[i] = comb;
        return comb;
    }
```
