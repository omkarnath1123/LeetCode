There are `n` houses in a village. We want to supply water for all the houses by building wells and laying pipes.

For each house `i`, we can either build a well inside it directly with cost `wells[i - 1]` (note the `-1` due to **0-indexing**), or pipe in water from another well to it. The costs to lay pipes between houses are given by the array `pipes` where each `pipes[j] = [house1j, house2j, costj]` represents the cost to connect `house1j` and `house2j` together using a pipe. Connections are bidirectional, and there could be multiple valid connections between the same two houses with different costs.

Return _the minimum total cost to supply water to all houses_.

**Example 1:**

![](https://assets.leetcode.com/uploads/2019/05/22/1359_ex1.png)
```
Input: n = 3, wells = [1,2,2], pipes = [[1,2,1],[2,3,1]]
Output: 3
Explanation: The image shows the costs of connecting houses using pipes.
The best strategy is to build a well in the first house with cost 1 and connect the other houses to it with cost 2 so the total cost is 3.
```
**Example 2:**
```
Input: n = 2, wells = [1,1], pipes = [[1,2,1],[1,2,2]]
Output: 2
Explanation: We can supply water with cost two using one of the three options:
Option 1:
  - Build a well inside house 1 with cost 1.
  - Build a well inside house 2 with cost 1.
The total cost will be 2.
Option 2:
  - Build a well inside house 1 with cost 1.
  - Connect house 2 with house 1 with cost 1.
The total cost will be 2.
Option 3:
  - Build a well inside house 2 with cost 1.
  - Connect house 1 with house 2 with cost 1.
The total cost will be 2.
Note that we can connect houses 1 and 2 with cost 1 or with cost 2 but we will always choose the cheapest option. 
```
**Constraints:**
-   `2 <= n <= 104`
-   `wells.length == n`
-   `0 <= wells[i] <= 105`
-   `1 <= pipes.length <= 104`
-   `pipes[j].length == 3`
-   `1 <= house1j, house2j <= n`
-   `0 <= costj <= 105`
-   `house1j != house2j`

## Detailed Intuition

### 💡 How to Think About the Solution

1. **Graph Transformation Concept:**
   - Convert problem into a Minimum Spanning Tree (MST)
   - Wells can be thought of as edges from a virtual source
   - Each pipe is an edge between houses
   - Total cost = sum of chosen edges

2. **Key Insights:**
   - Add virtual node (node 0) representing water source
   - Connect node 0 to each house with well cost
   - Use Kruskal's or Prim's algorithm for MST
   - Final MST gives optimal water distribution

### 🔍 Edge Cases to Consider
- Only two houses
- No pipes between houses
- Multiple pipes between same houses
- Very expensive wells vs cheap pipes
- Very expensive pipes vs cheap wells
- Maximum number of houses (10^4)
- Maximum cost (10^5)
- Disconnected components
- Cycle in pipe connections

### 🎯 Solution Approaches

1. **Kruskal's Algorithm with UnionFind**
   ```pseudo
   function minCostToSupplyWater(n, wells, pipes):
       edges = []
       # Add virtual edges for wells
       for i in range(n):
           edges.append((wells[i], 0, i+1))
       
       # Add all pipes
       for pipe in pipes:
           edges.append((pipe[2], pipe[0], pipe[1]))
       
       # Sort by cost
       sort edges
       
       uf = UnionFind(n+1)  # +1 for virtual node
       totalCost = 0
       
       for cost, u, v in edges:
           if uf.union(u, v):
               totalCost += cost
       
       return totalCost
   ```
   Time: O(E log E), Space: O(V)
   where E = pipes + n, V = houses + 1

## Detailed Intuition

- Model the problem as nodes and edges, then choose BFS/DFS/Topological traversal based on dependency direction.
- Track visited/state to prevent repeated processing and cycles.
- Build the answer while traversing (order, count, or connectivity result).

**Time Complexity** : O(e log(e))<br>
**Space Complexity** : O(n)

```java
class Solution {
    class UnionFind {
        int[] parent;
        int[] rank;
        
        public UnionFind(int n) {
            parent = new int[n];
            rank = new int[n];
            for (int i = 0; i < n; i++)
                parent[i] = i;
        }
        
        public int find(int x) {
            if (parent[x] != x)
                parent[x] = find(parent[x]);
            return parent[x];
        }
        
        public boolean union(int x, int y) {
            int px = find(x), py = find(y);
            if (px == py) return false;
            
            if (rank[px] < rank[py])
                parent[px] = py;
            else if (rank[px] > rank[py])
                parent[py] = px;
            else {
                parent[py] = px;
                rank[px]++;
            }
            return true;
        }
    }
    
    public int minCostToSupplyWater(int n, int[] wells, int[][] pipes) {
        List<int[]> edges = new ArrayList<>();
        
        // Add virtual edges for wells
        for (int i = 0; i < n; i++)
            edges.add(new int[]{0, i + 1, wells[i]});
        
        // Add all pipes
        for (int[] pipe : pipes)
            edges.add(pipe);
        
        // Sort by cost
        edges.sort((a, b) -> Integer.compare(a[2], b[2]));
        
        UnionFind uf = new UnionFind(n + 1);
        int totalCost = 0;
        
        for (int[] edge : edges) {
            if (uf.union(edge[0], edge[1]))
                totalCost += edge[2];
        }
        
        return totalCost;
    }
}
```
