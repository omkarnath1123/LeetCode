You are given `n` balloons, indexed from `0` to `n - 1`. Each balloon is painted with a number on it represented by an array `nums`. You are asked to burst all the balloons.

If you burst the `ith` balloon, you will get `nums[i - 1] * nums[i] * nums[i + 1]` coins. If `i - 1` or `i + 1` goes out of bounds of the array, then treat it as if there is a balloon with a `1` painted on it.

Return _the maximum coins you can collect by bursting the balloons wisely_.

**Example 1:**
```
Input: nums = [3,1,5,8]
Output: 167
Explanation:
nums = [3,1,5,8] --> [3,5,8] --> [3,8] --> [8] --> []
coins =  3*1*5    +   3*5*8   +  1*3*8  + 1*8*1 = 167
```
**Example 2:**
```
Input: nums = [1,5]
Output: 10
```
**Constraints:**
-   `n == nums.length`
-   `1 <= n <= 300`
-   `0 <= nums[i] <= 100`

## Detailed Intuition

- Define DP state so each state captures the minimum information required to continue transitions.
- Write transition from previously solved states and compute in valid order.
- Initialize base cases carefully since all later states depend on them.

**Time Complexity** : O(n<br>
**Space Complexity** : O(n

```java
    public int maxCoins(int[] nums) {
        int[][] store = new int[nums.length][nums.length];
        return maxCoins(nums, 0, nums.length - 1, store);
    }
    private int maxCoins(int[] nums, int start, int end, int[][] store) {
        if (start > end)
            return 0;
        if (store[start][end] != 0) return store[start][end];
        
        int max = 0;
        for (int i = start; i <= end; i++) {
            int val = getValue(nums, start - 1) * getValue(nums, end + 1) * getValue(nums, i);
            val = val + maxCoins(nums, start, i - 1, store);
            val = val + maxCoins(nums, i + 1, end, store);
            max = Math.max(val, max);
        }
        store[start][end] = max;
        return max;
    }
    private int getValue(int[] nums, int i) {
        if (i == -1 || i == nums.length) return 1;
        return nums[i];
    }
```
