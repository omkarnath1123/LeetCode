You are given an `m x n` integer matrix `grid`​​​, where `m` and `n` are both **even** integers, and an integer `k`.

The matrix is composed of several layers, which is shown in the below image, where each color is its own layer:

![](https://assets.leetcode.com/uploads/2021/06/10/ringofgrid.png)

A cyclic rotation of the matrix is done by cyclically rotating **each layer** in the matrix. To cyclically rotate a layer once, each element in the layer will take the place of the adjacent element in the **counter-clockwise** direction. An example rotation is shown below:

![](https://assets.leetcode.com/uploads/2021/06/22/explanation_grid.jpg)

Return _the matrix after applying_ `k` _cyclic rotations to it_.

## Detailed Intuition

- Identify the target relation for each index/value and scan once with supporting data structures if needed.
- Keep updates local and avoid recomputing previous work.
- Validate edge cases such as duplicates, empty ranges, and boundary indices.

**Time Complexity** : O(n)<br>
**Space Complexity** : O(1)

```java
    public int[][] rotateGrid(int[][] grid, int k) {

        int len = grid.length,
            bre = grid[0].length;

        for (int i = 0; i < Math.min(len / 2, bre / 2); i++) {

            int count = 2 * (len - 2 * i) + 2 * (bre - 2 * i) - 4;
            if (k % count == 0) continue;
            for (int j = 0; j < k % count; j++)
                rotateOne(grid, i, count, len, bre);
        }
        return grid;
    }
    int[][] dirs = new int[][]{{0, 1}, {1, 0}, {0, -1}, {-1, 0}};
    private void rotateOne(int[][] grid, int depth, int count, int len, int bre) {

        int temp = 0,
                x = depth,
                y = depth,
                pos = 0,
                p = 0,
                q = 0;
        for (int i = 0; i < count - 1; i++) {
            p = x + dirs[pos][0];
            q = y + dirs[pos][1];

            if (p < depth || q < depth || p >= len - depth || q >= bre - depth) {
                pos++;
                p = x + dirs[pos][0];
                q = y + dirs[pos][1];
            }

            temp = grid[p][q];
            grid[p][q] = grid[x][y];
            grid[x][y] = temp;
            x = p;
            y = q;
        }
    }
```
