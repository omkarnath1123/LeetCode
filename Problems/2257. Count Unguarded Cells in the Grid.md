You are given two integers `m` and `n` representing a **0-indexed** `m x n` grid. You are also given two 2D integer arrays `guards` and `walls` where `guards[i] = [rowi, coli]` and `walls[j] = [rowj, colj]` represent the positions of the `ith` guard and `jth` wall respectively.

A guard can see **every** cell in the four cardinal directions (north, east, south, or west) starting from their position unless **obstructed** by a wall or another guard. A cell is **guarded** if there is **at least** one guard that can see it.

Return _the number of unoccupied cells that are **not** **guarded**._

**Constraints:**

- `1 <= m, n <= 105`
- `2 <= m * n <= 105`
- `1 <= guards.length, walls.length <= 5 * 104`
- `2 <= guards.length + walls.length <= m * n`
- `guards[i].length == walls[j].length == 2`
- `0 <= rowi, rowj < m`
- `0 <= coli, colj < n`
- All the positions in `guards` and `walls` are **unique**.

**Solution:**

- [x] [[Hash Table]]

## Detailed Intuition

- Model the problem as nodes and edges, then choose BFS/DFS/Topological traversal based on dependency direction.
- Track visited/state to prevent repeated processing and cycles.
- Build the answer while traversing (order, count, or connectivity result).

**Time Complexity** : O(n)<br>
**Space Complexity** : O(n)

```java
    int[][] dirs = new int[][]{
        {-1, 0}, {1, 0}, {0, 1}, {0, -1}
    };
    public int countUnguarded(int m, int n, int[][] guards, int[][] walls) {
        
        Set<Point> wallsPos = new HashSet<>(),
                    guardsPos = new HashSet<>(),
                    guarded = new HashSet<>();

        for (int[] guard : guards)
            guardsPos.add(new Point(guard[0], guard[1]));
        for (int[] wall : walls)
            wallsPos.add(new Point(wall[0], wall[1]));
        
        for (Point guardPos : guardsPos)
            for (int[] dir : dirs) {
                
                Point next = new Point(guardPos.x + dir[0], guardPos.y + dir[1]);
                while (next.x >= 0 && 
                        next.y >= 0 && 
                        next.x < m && 
                        next.y < n && 
                        !guardsPos.contains(next) && 
                        !wallsPos.contains(next)) {
                            guarded.add(new Point(next.x, next.y));
                            next.addNext(dir[0], dir[1]);
                }
            }
        
        return (int)((long) m * (long) n - wallsPos.size() - guardsPos.size() - guarded.size());
    }
    class Point {
        int x, y;
        public Point(int x, int y) {
            this.x = x;
            this.y = y;
        }
        public void addNext(int p, int q) {
            this.x = this.x + p;
            this.y = this.y + q;
        }
        @Override
        public boolean equals(Object obj) {
            Point that = (Point) obj;
            return this.x == that.x && this.y == that.y;
        }
        @Override
        public int hashCode() {
            return Objects.hash(x, y);
        }
    }
```
