Given an binary array `nums` and an integer `k`, return `true` _if all_ `1`_'s are at least_ `k` _places away from each other, otherwise return_ `false`.

**Example 1:**

![](https://assets.leetcode.com/uploads/2020/04/15/sample_1_1791.png)

```
Input: nums = [1,0,0,0,1,0,0,1], k = 2
Output: true
Explanation: Each of the 1s are at least 2 places away from each other.
```

**Example 2:**

![](https://assets.leetcode.com/uploads/2020/04/15/sample_2_1791.png)

```
Input: nums = [1,0,0,1,0,1], k = 2
Output: false
Explanation: The second 1 and third 1 are only one apart from each other.
```

**Constraints:**

- `1 <= nums.length <= 105`
- `0 <= k <= nums.length`
- `nums[i]` is `0` or `1`

**Solution:**

- [x] [[Array]]

## Detailed Intuition

- Keep track of the last position where a `1` was found.
- Every time a new `1` appears, check if the gap satisfies the requirement.
- If the gap is too small, the condition fails.

**Time Complexity** : O(n)<br>
**Space Complexity** : O(1)

```java
    public boolean kLengthApart(int[] nums, int k) {
        
        int prevIndex = -1 * k - 1;
        for (int index = 0; index < nums.length; index++) {
            if (nums[index] == 1) {
                if (index - prevIndex - 1 < k) return false;
                prevIndex = index;
            }
        }
        return true;
    }
```
